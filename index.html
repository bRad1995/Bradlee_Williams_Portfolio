Turning Point UPC

Django Application

Turning Point UPC is an application built for the purpose of helping a local congregation showcase thier organization on a sophisticated platform. It is an informational website that provides general information about the church's staff members, contact information and scheduled events. The app features administrative functions and access thanks to the help of Django.

Technologies Used: Django, Python, PostgreSQL, Neon, Bootstrap, HTML and CSS

React Application

devPort is a social platform designed for software developers to showcase thier work and connect with other developers. The idea for this project was to create something that was a "Instagram-LinkedIn hybrid", but with a more causal approach. This was my first group project, working with 3 other cohorts. I performed as our Branch Manager and assitant backend engineer.

Smoky-Trout Streams

EJS Application

Smoky-Trout Streams is an informational website with the intent of being a helpful resource to fly fishermen in the Great Smoky Mountain region of Tennessee. The site offers descriptive information on stream locations, their fishability and what equipment to use.

Technologies Used: Express, NodeJS, Mongoose, HTML and CSS.

Simon's REVENGE

JavaScript Game

Simon's Revenge is a JavaScript game based off of the original "Simon" from Hasbro. The object of the game is simple. The game will flash a certain sequence of buttons randomly and then the player must mimic the pattern correctly in order to proceed to the next level. If the player fails to match the game's pattern, the game will reset to it's original starting point.

Technologies Used: JavaScript, CSS and HTML.
